# Network Toolkit Triage

This folder supports the Network Toolkit Triage workflow.

- `Tools` is for optional portable triage utilities that are referenced by `App\manifests\triage-tools.json`.
- `Profiles` is reserved for future named collection profiles.
- `Runs` stores per-computer triage runs. Each run contains collected files, command output, event summaries, local analysis, metadata, and a ZIP bundle.
- `Templates` is reserved for future prompt and report templates.

Triage runs can contain client data. Use the toolkit sanitation controls before sharing or archiving a toolkit copy.
